# A股市场综合评估技能

一个 Cursor Agent Skill，用于生成专业的 A 股市场综合评估报告。

## 功能

基于四大核心指标，输出：
- **综合评级**：从"绝望"到"极度泡沫"的 10 级评级
- **综合评分**：0-100 分的量化评估
- **详细分析**：每个指标的数据解读
- **投资建议**：针对不同风险偏好的具体策略
- **风险提示**：当前市场的主要风险点

## 五大核心指标 + 宽度修正

| 指标 | 权重 | 数据来源 |
|------|------|---------|
| 市盈率 | 25% | 指数PE、行业PE、历史分位 |
| 两融余额 | 20% | 融资融券余额、占比、分位 |
| IPO数量 | 10% | 发行数量、募资额、新股表现 |
| 风险溢价 | 25% | 股票预期收益率 vs 国债收益率 |
| **盈利周期** | **20%** | 工业利润增速、M1、社融（⭐新增） |
| 宽度修正 | 辅助 | 涨跌家数比、均线占比 |

## 评级体系

| 评分区间 | 评级 | 市场状态 |
|----------|------|---------|
| 90-100 | 极度泡沫 | 随时崩盘风险 |
| 80-90 | 泡沫 | 多数指标过热 |
| 70-80 | 狂热 | 投机氛围浓厚 |
| 60-70 | 乐观 | 偏热但可参与 |
| 50-60 | 谨慎乐观 | 中性偏暖 |
| 40-50 | 中性 | 多空平衡 |
| 30-40 | 谨慎 | 偏冷观望 |
| 20-30 | 悲观 | 情绪低迷 |
| 10-20 | 恐慌 | 非理性抛售 |
| 0-10 | 绝望 | 极度低估（黄金坑） |

## 安装

将整个 `a-share-market-assessment` 文件夹复制到：

```
macOS/Linux: ~/.cursor/skills/
Windows: %USERPROFILE%\.cursor\skills\
```

## 使用

在 Cursor 中用自然语言提问即可触发：

### 买卖决策类
- "A股该买还是该卖"
- "现在该加仓还是减仓"
- "该不该抄底"

### 市场状态类
- "现在是牛市还是熊市"
- "A股目前处于什么状态"
- "现在是顶部还是底部"

### 分析评估类
- "帮我分析下A股"
- "大盘现在怎么样"
- "股市风险大吗"

### 仓位建议类
- "现在该持有多少仓位"
- "满仓安全吗"

### 示例输入

```
A股该买还是该卖？
```

```
现在是牛市还是熊市？
```

```
帮我分析下A股目前的状态
```

```
├── reference.md    # 参考数据（历史阈值、行业基准）
├── examples.md     # 示例输出（不同市场状态）
└── README.md       # 本文件
```

## 注意事项

1. **数据时效性**：需要获取实时/近期市场数据才能生成准确报告
2. **投资建议免责**：报告仅供参考，不构成投资建议
3. **结构性分析**：注意区分整体与结构，避免"一刀切"结论
