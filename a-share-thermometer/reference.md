## 9. 常用公式

### 市场预期收益率
```
方法1：市场预期收益率 ≈ 1 / 市场PE × 100%
方法2：市场预期收益率 ≈ 股息率 + 预期盈利增长率
```

### 风险溢价（利差）
```
风险溢价 = 市场预期收益率 - 10年期国债收益率
```

### 综合评分
```
综合评分 = PE得分×0.30 + 两融得分×0.25 + IPO得分×0.15 + 利差得分×0.30
```
