# A股温度计命令

当用户输入以下命令时，立即读取并执行 A股温度计技能：

- `/A股温度计`
- `/温度计`
- `/a股`
- `/ashare`

## 执行流程

1. 识别到上述命令后，读取技能文件：`~/.cursor/skills/a-share-thermometer/SKILL.md`
2. 按照技能定义的流程执行：
   - 判断目标交易日（已收盘评估当天，未收盘评估上一交易日）
   - 获取市场数据（上证指数、两融余额、涨跌家数等）
   - 计算五维指标得分
   - 应用市场宽度修正
   - 输出温度计报告

## 输出格式

必须包含：
- 🌡️ 温度可视化（0-100°）
- 五维体检表格
- 操作建议（分投资者类型）
