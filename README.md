# A股温度计 🌡️

量化评估 A 股市场的冷热程度，像温度计一样直观显示市场状态。

## 功能

基于五大核心指标 + 市场宽度修正，输出：
- **市场温度**：0-100°（🧊冰点 → 🔥沸腾）
- **状态评级**：从"绝望"到"极度泡沫"的 10 级
- **五维体检**：估值、杠杆、IPO、利差、盈利周期
- **投资建议**：针对不同风险偏好的具体策略
- **风险预警**：当前市场的主要风险点

## 五大核心指标 + 宽度修正

| 指标 | 权重 | 数据来源 |
|------|------|---------|
| 市盈率 | 25% | 指数PE、行业PE、历史分位 |
| 两融余额 | 20% | 融资融券余额、占比、分位 |
| IPO数量 | 10% | 发行数量、募资额、新股表现 |
| 风险溢价 | 25% | 股票预期收益率 vs 国债收益率 |
| **盈利周期** | **20%** | 工业利润增速、M1、社融（⭐新增） |
| 宽度修正 | 辅助 | 涨跌家数比、均线占比 |

## 温度体系 🌡️

| 温度 | 状态 | 操作建议 |
|------|------|---------|
| 90-100° | 🔥 沸腾 | 清仓观望 |
| 80-90° | 🥵 极热 | 大幅减仓 |
| 70-80° | 😰 偏热 | 控制仓位 |
| 60-70° | 😊 温暖 | 精选个股 |
| 50-60° | 😐 适中 | 均衡配置 |
| 40-50° | 🤔 微凉 | 逢低布局 |
| 30-40° | 😟 偏冷 | 分批建仓 |
| 20-30° | 😨 寒冷 | 积极布局 |
| 10-20° | 🥶 极寒 | 大胆加仓 |
| 0-10° | 🧊 冰点 | 满仓抄底 |

## 安装

将整个 `a-share-market-assessment` 文件夹复制到：

```
macOS/Linux: ~/.cursor/skills/
Windows: %USERPROFILE%\.cursor\skills\
```

## 使用

在 Cursor 中用自然语言提问即可触发：

### 买卖决策类
- "A股该买还是该卖"
- "现在该加仓还是减仓"
- "该不该抄底"

### 市场状态类
- "现在是牛市还是熊市"
- "A股目前处于什么状态"
- "现在是顶部还是底部"

### 分析评估类
- "帮我分析下A股"
- "大盘现在怎么样"
- "股市风险大吗"

### 仓位建议类
- "现在该持有多少仓位"
- "满仓安全吗"

### 示例输入

```
A股该买还是该卖？
```

```
现在是牛市还是熊市？
```

```
帮我分析下A股目前的状态
```


## 实时数据获取

技能内置 `fetch_market_data.py` 脚本，自动从API获取实时数据：

**数据源**：
- 腾讯财经API（优先）
- 东方财富API（备用）

**测试脚本**：
```bash
python3 ~/.cursor/skills/a-share-thermometer/fetch_market_data.py
```

## 注意事项

1. **数据时效性**：技能会自动调用脚本获取实时数据，确保准确性
2. **投资建议免责**：报告仅供参考，不构成投资建议
3. **结构性分析**：注意区分整体与结构，避免"一刀切"结论
4. **依赖要求**：需要系统安装 `curl` 和 `python3`（macOS/Linux 通常已自带）
